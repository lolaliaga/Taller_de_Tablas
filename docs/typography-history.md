# Typography history

## Before switching to Barlow Condensed

The project previously used **Inter** as the primary typeface.

- **Loaded in:** `reparaciones/templates/reparaciones/base.html`
- **CSS usage:** `reparaciones/static/reparaciones/estilos.css`
- **Weights:** 400, 500, 600, 700

### Legacy references

- Google Fonts import:
  ```html
  <link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700&display=swap" rel="stylesheet">
  ```
- CSS declarations:
  ```css
  body {
    font-family: 'Inter', sans-serif;
  }

  input,
  select,
  textarea,
  button,
  label {
    font-family: 'Inter', sans-serif;
  }
  ```
